獨居小前奏

Posté le 2017-04-28 | In ethics | | Visitors:

2017.4.28

"你比我想象中的还要更学生气些，太腼腆了"，这是一个刚巧也住在公寓的有几面之缘的90年小哥在这次面基开始后没多久的的评价，尽管我还穿着没来得及换下的正装和皮鞋，因为加班且下雨先把快递寄过来的鞋架和路由器送回家之后没换便装就下楼来了，前几天小哥没空所以没约到但今天刚好可以见一面。约面的目的很简单，一是熟悉下公寓的一些情况更好的掌握到一些先验信息二是好久不见也算混个面熟。但两句话着实给我一个激灵，用最近流行的家乡话讲真是扎心了老铁！后来他也补充道社会经验不足也还好毕竟"你还小"，可以说"你还小"我已经听了不下十遍了。
扎心是因为这的确是我自己也意识到的问题但解决起来一直没什么进展或者说没有到令我满意的程度，不知道是自己太着急了还是自己真是个失败者，因为不愿意承认自己是个失败者吧，至少思想上或是信息获取上还有一点超越同龄人的优越感。给自己的束缚太多了法律也好道德也好总想做一个正人君子严格要求自己？该不该好不好值不值深究起来就太复杂了，但是但是毕竟要开始一个人的独居生活了，希望能在这短则一年长达数年的阶段破茧而出，结果是死亡或化蝶都无所谓。
we want to find love, we want to success, until nothing was enough…
