致高中時代

Posted on 2019-02-03 | In ethics | | Visitors:

2011.08.19

三年的高中生活让我成熟许多，即将踏入大学的象牙塔，在此杀青高中，开启大学吧！

进入七班，无疑是我最幸运的事之一了。我们清晰地记得语文老师的“回眸一笑…”数学老师的“五花八门，什么都有…”英语老师的“开学第一堂纯英语授课…”物理老师的“从一中前门到佳斯特…”化学老师的“无敌讲课45分钟不停歇”生物老师的“理智又清晰的问答”,他/她们八仙过海,各显神通,能成为他/她们的学生,我很自豪，真的非常感谢他/她们.当然能和各位成为同学我也很开心，七班这个大家庭很有凝聚力。

其实我感觉自己性格有点怪，或许是因为我的经历吧。每个人都有自己的经历，经历又造就了性格，七班每个人都有自己的个性，只不过有的人没表现出来罢了。我知道有时我伤到了某些人，甚至扫了大家的兴,实在是对不住各位了，请见谅，我会改的,谢谢你们的宽容。一日不见，当刮目相看，再见面时，再挑挑我的毛病吧，呵呵。

当然还要感谢一直关心支持我的亲人朋友和其他老师同学们,我很幸运也很幸福!

不过有点遗憾的是没在校园舞台上唱过歌,没在校报上发表过文章,没谈过恋爱…还好有点个人爱好,不然真被中国式教育给抹杀了.

说实话,我是长得着急了点,胡茬一把,不过咱今年才18周岁啊,我年龄的4次方和6次方可囊括了所有数字了呢!各位复读的和升学的兄弟姐妹,发仔的小宇宙可要爆发了,不过还得等几年才能看到疗效,看你们的啦！

还记得2011年元旦我唱的那首歌吗？（再见-张震岳）

春天绽放的花蕊 夏季里更加明媚 秋天的落叶 冬天的风雪 我们都一起体会 经过四季的轮回 学会要勇敢面对

今天的我们更相信 笑容之后的眼泪 这里是我们共同的家 有我们最美的年华 年轻的梦在这里发芽 走过了春秋冬夏 明天又开始新的出发 请不要担心害怕 告别了青春的美丽童话 我们都已经长大。。。加油吧！！！

尾记：花样年华，似水流年。写不休的更感，奏不明的更叹，恰便是那遮不住的青山隐隐，留不住的溪水潺潺。

短评：高中没啥好说的 从太不自信到不太自信 活在死气沉沉的评判标准里 感谢我活下来了
